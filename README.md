# ❤️ Sasco_Tech's Portfolio 


## 🔥 Clone this Repository
You need to write the following commands on the terminal screen(in vscode) so that you can run this project locally.

```bash
  git clone "https://github.com/Saleh-abdulsamad11/my_portfolio.git "
```
Go to the project directory

```bash
  cd Sasco_portfolio
```
Install dependencies
```bash
  npm install
```
Start the server
```bash
  npm start
```

This application will now be running on `localhost`.
